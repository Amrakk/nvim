# nvim

Neovim configuration repository with custom features.

## Features

### U Pattern Feature
This configuration includes a custom feature that implements the "uuuuuuuuuuu" pattern.

**Usage:**
- Press `<leader>u` in normal mode to insert the pattern "uuuuuuuuuuu"

## Installation

1. Clone this repository to your Neovim config directory:
   ```bash
   git clone https://github.com/Amrakk/nvim ~/.config/nvim
   ```

2. Start Neovim and the configuration will be loaded automatically.

## Configuration

The main configuration is in `init.lua` which includes:
- Basic editor settings (line numbers, indentation)
- Custom u pattern insertion function
- Key mapping for quick pattern insertion
